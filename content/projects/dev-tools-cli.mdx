# Dev Tools CLI

A command-line toolkit for automating repetitive dev workflows — scaffolding, code generation, and deployment helpers.

## Features

- **Project Scaffolding** — Quick setup for new projects
- **Code Generation** — Generate boilerplate and common patterns
- **Deployment Helpers** — Streamlined deployment workflows

## Tech Stack

- Node.js
- TypeScript
- Bun

## Status

Currently a work in progress. Planning to add more utilities as needs arise.
