# Portfolio

My personal portfolio built with Next.js 16, Tailwind CSS v4, and next-themes. Fully data-driven — all content lives in JSON files.

## Features

- **Data-driven** — All content (projects, experience, skills, certifications) lives in JSON files
- **Dark mode** — Automatic theme switching with system preference detection
- **Responsive** — Optimized for all screen sizes
- **Fast** — Static generation with Next.js App Router

## Tech Stack

- Next.js 16 with App Router
- Tailwind CSS v4
- TypeScript
- Vercel for hosting

## What I Learned

Building this portfolio taught me a lot about modern web development patterns, especially around data-driven architectures and theme management.
