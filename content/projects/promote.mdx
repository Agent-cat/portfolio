# Promote

A platform for creating and managing promotional campaigns. Built the complete frontend infrastructure from scratch including design system and API integration.

## Features

- **Campaign Management** — Create, edit, and manage promotional campaigns
- **Design System** — Consistent UI components across the platform
- **API Integration** — Efficient data fetching with error handling
- **Performance** — Optimized for speed and scalability

## Tech Stack

- Next.js with TypeScript
- Tailwind CSS
- Figma for design
- Turborepo for monorepo management

## My Role

As the Founding Frontend Engineer, I architected the complete frontend infrastructure and led codebase refactoring initiatives.
