# Samyak 2025 — Event & Fest Management Platform

- Built a full-stack event management system using **React.js**, **Node.js**, **Redis**, and **Multer**, enabling streamlined participant and organizer workflows.
- Improved backend reliability through security middleware, validation layers, and API rate limiting, ensuring stable performance during peak event traffic.
- Developed an intuitive dashboard enabling organizers to manage participants, track activity, and distribute certificates in real-time.
