# Taskr

Minimal task management app with real-time sync, drag-and-drop boards, and team collaboration features.

## Features

- **Real-time Sync** — Changes sync instantly across devices
- **Drag & Drop** — Intuitive board-based task management
- **Team Collaboration** — Share boards with team members

## Tech Stack

- React
- Supabase for backend and real-time
- Tailwind CSS
- TypeScript

## Status

Archived — Built as a learning project to explore real-time collaboration patterns.
